Code Golf

What is Code Golf?

The idea behind code golf is simple. Write a program to complete the given challenge in any language you like: the shortest correct code wins!

The Challenge

Given an integer n, find the next integer m with the same number of 1s in its binary expansion as n’s

For example, for n = 3 (dec) = 11 (bin), the next integer m with two 1s in its binary expansion is 101 (bin) = 5 (dec)

Input (from stdin): The decimal representation of n
Output (to stdout): The decimal representation of m
Constraints: 0 < n < 1000

Submission:

Send your code to @pbsinclair42 on Slack along with your name and the language (and version) used.

If you’ve any questions, feel free to drop me a message.
